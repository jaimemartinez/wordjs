Para correr el proyecto WordJS, necesitas iniciar dos servidores por separado: el backend (la API y el sitio principal) y el frontend (el panel de administración).

Aquí tienes los comandos divididos por sección:

1. Backend (API y Sitio Web)
Desde la carpeta raíz del proyecto (wordjs):

powershell
npm run dev
URL: http://localhost:3000
API: http://localhost:3000/api/v1
2. Frontend (Panel de Administración)
Debes entrar en la carpeta admin-next e iniciar el servidor de Next.js:

powershell
cd admin-next
npm run dev
URL: http://localhost:3001/login
Nota: El comando internamente usa el puerto 3001 para no entrar en conflicto con el backend.
